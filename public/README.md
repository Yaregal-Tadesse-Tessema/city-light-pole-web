# Public Assets Folder

This folder contains static assets that are served directly by Vite.

## Logo File

Please place the ESAA logo file here with the filename: `esaa-logo.jpeg`

The logo should be:
- A circular logo with gold border
- Blue background with white lightbulb and handshake graphic
- ESAA acronym in gold letters
- Recommended size: 120x120 pixels or larger (will be scaled down)

The logo will be displayed in the sidebar/navbar of the application.
